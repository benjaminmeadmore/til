 # The management JSON file 

Using a file called the `admin-settings.json` (you can specify and then lock the configuration parameters when creating a standardized Docker Dekstop env across your org or userspace. 

 ## admin-settings.json

 ```zsh
 #admin-settings.json location
/Library/Application\ Support/com.docker.docker/admin-settings.json
```
This file either needs to be created manually in the above location, or can be created on startup when installing with the `--admin-settings` flag.
